E-Commerce Sales Dashboard

Welcome! This is my interactive E-Commerce Sales Dashboard project, created as a part of my learning journey in Data Analytics & Business Intelligence.
As a fresher, I designed this dashboard to explore real-world style e-commerce data and convert it into meaningful insights.

🔹 Project Overview

This dashboard provides a comprehensive view of e-commerce sales, profit trends, customer behavior, and regional performance. It helps businesses make data-driven decisions and understand key performance metrics visually.

📊 Key Features
Total Sales & Profit: Quick overview of business performance.
Monthly Sales Trend: Track how sales evolve month by month.
Profit Analysis: Visualize monthly profit and profit margin by category.
Top Customers & Cities: Identify high-value customers and revenue-generating locations.
Orders by Payment Mode: See which payment methods are most popular.
Interactive Filters: Analyze data by category, city, or quarter.
Repeat Customer Insights: Understand customer retention trends.
🛠 Tools & Techniques Used
Power BI: Dashboard creation & visualization
Microsoft Excel: Data cleaning & preparation
DAX Basics: Calculations & metrics
Data Visualization: Charts, KPIs, and interactive reports
🎯 Objective

The main goal of this project is to turn raw e-commerce data into actionable insights.
It demonstrates how analytics and BI tools can help businesses improve decision-making and monitor performance effectively.

🚀 Learning Outcome
Built skills in data cleaning, visualization, and dashboard designing.
Learned to create interactive, user-friendly reports.
Gained experience in applying real-world business scenarios to analytics.

This project is a stepping stone in my journey as a Data Analytics & BI professional, and I am continuously exploring and learning new techniques to improve my skills.
